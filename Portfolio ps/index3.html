What are Tag Selectors?

Tag selectors (also called element selectors) target all HTML elements of a specific type. They are the most basic type of CSS selector and apply styles globally to all instances of that tag throughout the page.

Key Characteristics

Tag selectors have the lowest specificity among the three main selector types. They are perfect for setting default styles for your entire website or application.

10 Tag Selectors Demonstrated

  • body - Styles the entire page background and font
  • h1, h2, h3 - Styles all heading elements
  • p - Styles all paragraph text
  • ul & li - Styles all lists and list items
  • a - Styles all anchor links
  • button - Styles all button elements
  • section & article - Styles semantic containers
  • table, th, td - Styles table elements

Interactive Example

Notice how every paragraph on this page has the same styling? That's because we used the p tag selector. Similarly, all links, buttons, and headings share consistent styling.

Comparison Table

Selector Type Syntax Specificity Use Case
Tag Selector p { } Low Global styles
Class Selector .classname { } Medium Reusable styles
ID Selector #idname { } High Unique elements

Why Use Tag Selectors?

Tag selectors are excellent for establishing a consistent baseline style across your entire website. They ensure that all elements of a certain type have a uniform appearance unless overridden by more specific selectors.

Best Practices

Use tag selectors for general styling and typography. Combine them with class and ID selectors for more specific styling needs. Remember the CSS cascade: more specific selectors will override tag selectors.
