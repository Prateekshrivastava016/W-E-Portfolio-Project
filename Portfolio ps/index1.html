What are ID Selectors?

ID selectors are used to target a single, unique element on the page. They have high specificity and override class and tag selectors. IDs are defined with the # symbol in CSS.

This text is styled using #highlight-text ID selector!

  • High specificity in CSS cascade
  • Should be unique per page
  • Perfect for JavaScript targeting
  • Used for landmark elements
